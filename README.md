# WCPE

世界那么大，组件那么小。Wcpe，做Android Wear和Android Phone之间的通讯框架。

[![License Apache2.0](https://img.shields.io/hexpm/l/plug.svg)][license]


支持平台：[Android](Android) API 4.3+  

## 功能
* **支持Wear到Phone的通讯**
  - 服务器客户端模式
* **Wear上的图片加载器**
  - 异步加载图片
  - 本地缓存
